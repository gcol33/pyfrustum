# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-17

### Added

- Initial release
- Core geometry primitives: Mesh, PointCloud, Polyline
- Solid and scalar-mapped materials
- Perspective and orthographic cameras
- Directional lighting with presets
- Marching cubes isosurface extraction
- Multi-isosurface extraction
- Volume operations: gradient magnitude, Laplacian, normalization
- GPU-accelerated rendering via wgpu
- PNG output
